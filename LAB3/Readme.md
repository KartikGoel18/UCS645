# UCS645 (Parallel and Distributed Computing)

## Important:
-   Use g++ version >= 15.0
-   Code is written in C++23.
-   This program is calculating the correlation matrix in both sequential and parallel methods.
  
-   To Compile this program open the command panel and use the following command :-
    ``` bash
    make
    ```
-   To Run Parallel open the command panel and use the following command :-
    ``` bash
    make run
    ```
-   To Run Parallel with custom matrix size open the command panel and use the following command :-
    ``` bash
    make run ARGS="<number of rows> <number of columns>"
    ```
-   To Run Sequential open the command panel and use the following command :-
    ``` bash
    make run-seq
    ```
-   To Run Sequential with custom matrix size open the command panel and use the following command :-
    ``` bash
    make run-seq ARGS="<number of rows> <number of columns>"
    ```
